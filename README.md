# Time Tracker – учёт рабочего времени сотрудников

Веб‑приложение для учёта времени, затраченного сотрудниками на проекты. Реализовано на Flask и SQLite.

## Возможности

- **Добавление записи**: имя сотрудника, дата, количество часов, название проекта.
- **Список всех записей** с возможностью редактирования и удаления.
- **Фильтрация записей по диапазону дат**.
- **Отчёт по сотруднику**: суммарные часы за период (или за всё время) и распределение по проектам.
- **Отчёт по проекту**: суммарные часы по проекту и список работавших сотрудников.
