# Time Tracker – учёт рабочего времени сотрудников

Веб‑приложение для учёта времени, затраченного сотрудниками на проекты. Реализовано на Flask и SQLite.

## Основные возможности

- **Учет записей**: Добавление, редактирование и удаление записей (сотрудник, дата, часы, проект).
- **Умный ввод**: Автозаполнение имен сотрудников и названий проектов при добавлении/изменении на основе уже сохраненных данных в базе.
- **Главная страница**: Общий список с фильтрацией по датам, быстрым подсчетом итогового количества отработанных часов и числа записей за выбранный период.
- **Визуальные отчёты (Chart.js)**: 
  - **Отчёт по сотруднику**: суммарное время, фильтр по датам и круговая диаграмма распределения времени по проектам.
  - **Отчёт по проекту**: суммарное время, фильтр по диапазону дат, список задействованных специалистов и диаграмма долей участия.
